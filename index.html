ORGANIISE.

Organiise is an application developed for Android, Organiise helps users keep track of what they have accomplished in a day. This is then visualised in a pie chart for the user to see.

IWORKOUT.

iWorkout is an application developed by android which helps users organise there workouts. Users can create there own workouts with a list of excercies, and then when users are exercising they can choose which one of there workouts they're performing. iWorkout helps users by calculating workout times based on the sets, and how much weight they are using compared to there last workout.

THE GAME SHOP.

The Game Shop is a CRUD (create, read, update and delete) program developed in c#.NET and implementing SQL databases that allows game stores to view there current stock, and allow them to add new stock in a quick and timley manner without needing a vast knowledge in databases.

CAR COSTER.

Car Coster is a windows application developed in C# that takes JSON files on the UK government website and converts them to Car objects, these car Objects contain information such as: the MPG of cars, the tax cost of cars, the total cost of cars, CO2 emissions and much more. Furthermore Car Coster can inform users about the most economical car of a given year (2000-2017) which can persuade the users to purchase said car.
